# 2026-05-12 /knowledge 工具栏视觉修正

## 背景

`/knowledge` 知识库列表页顶部工具栏在宽屏下横向撑满页面，搜索框和操作按钮与下方 1200px 主表格区域不对齐，视觉重心过宽。

## 调整

- 将工具栏卡片宽度限制为 `max-width: 1200px`，并居中对齐，与知识库表格保持一致。
- 将搜索输入从内联固定宽度改为 `.knowledge-search` 响应式样式，桌面端最大 360px，窄屏下铺满父容器。
- 保留原有搜索、刷新、新建知识库等交互逻辑，不改动接口与权限规则。

## 验证

- 前端样式检查：`npm run stylelint -- src/views/KnowledgeListView.vue`
